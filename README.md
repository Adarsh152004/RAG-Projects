# RAG Projects

A collection of hands-on **Retrieval-Augmented Generation (RAG)** projects built using **LangChain**, **LlamaIndex**, vector databases, and Large Language Models (LLMs).

## 📂 Projects

* 📄 PDF Question Answering
* 🎥 YouTube Video Chatbot
* 🌐 Website Question Answering
* 💬 Conversational RAG Chatbot
* 🔍 Document Retrieval
* 🧠 Vector Store Integrations
* 🚀 More RAG experiments coming soon...

## 🛠️ Tech Stack

* Python
* LangChain
* LlamaIndex
* OpenAI / Hugging Face
* FAISS / ChromaDB
* Streamlit
* Jupyter Notebook

## 🚀 Getting Started

```bash
git clone https://github.com/Adarsh152004/RAG-Projects.git
cd RAG-Projects
pip install -r requirements.txt
```

## 🎯 Purpose

This repository serves as a learning hub for experimenting with different RAG architectures, retrieval techniques, and GenAI applications.
